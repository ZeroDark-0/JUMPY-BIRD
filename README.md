\#  Flappy Bird — Unity 2D Clone



A fully playable \*\*Flappy Bird\*\* clone built in Unity 2D using C#, following the YouTube tutorial by @GMTK.  

Built as a hands-on introduction to Unity's 2D physics, sprite rendering, and game state management systems.



\---



\##  Gameplay



Navigate the bird through an endless series of pipes by tapping/clicking to flap.  

Each pipe pair you survive earns you a point. Hit a pipe or the ground — game over.

!\[preview](Flappy bird.gif)





\---



\##  Features



\-  Bird with gravity-based physics and flap mechanic

\-  Randomly spawning pipes with variable gap positions

\-  Infinite scrolling background and ground

\-  Game Over screen with restart functionality
